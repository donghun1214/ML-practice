# 간단한 주가예측 딥러닝 모델 (LSTM)

## 개발동기
RNN 기초를 배운 것을 미니프로젝트를 통해 써먹고 싶었다 

## 느낀 점
모델을 만들고, 파라미터 조정하는 것은 그렇게 어렵지 않다. LSTM 원리에 심층적으로 접근하면 말이 달라지겠지만, 모델을 사용하는 데는 크게 어렵지 않다. 
제일 시간이 많이 걸리는 것은 데이터 전처리다.
